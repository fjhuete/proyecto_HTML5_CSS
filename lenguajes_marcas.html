Lenguajes de Marcas

  • Proyecto JSON

    En este proyecto hemos trabajado con un fichero JSON para crear un programa en Python que consulte y modifique la información que contiene. En este caso se ha usado un fichero que contiene datos históricos de la tasa de paro agrupados por grupos de población y región geográfica.
